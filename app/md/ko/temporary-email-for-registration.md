## 가입용 임시 이메일

**가입용 임시 이메일**은 회원가입 시 기본 메일 주소를 노출하고 싶지 않을 때 유용합니다.

### 적합한 상황

- 체험 계정 생성
- 다운로드/쿠폰 인증
- 저위험 서비스 가입

### 사용 방법

1. smail.pw에서 일회용 주소 생성
2. 가입 폼에 주소 입력
3. OTP 또는 확인 링크 수신
4. 기본 메일 노출 없이 가입 완료
